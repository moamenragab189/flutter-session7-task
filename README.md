# Flutter Simple Product Detail Screen

A basic Flutter implementation of an e-commerce product detail page for a "Natural Red Apple." This is a straightforward learning project focused on practicing core Flutter widgets and layout concepts.

## Project Overview

This single-screen app includes:
- Product image and name
- Quantity selector with increment/decrement
- Price display and favorite icon
- Sections for product details, nutrition, and reviews (with 5-star rating)
- "Add to Basket" button
